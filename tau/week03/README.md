# 🧪 Tau Week03 – File Permissions & Users

## 🌐 Overview
Explore file permissions, users, and access control in Windows.

## 🧠 Why It Matters
Permissions protect sensitive files. Learning how they work is key to IT and security.

## 🔍 What You’ll Learn
- Use whoami and icacls to view permissions
- Understand user roles and groups
- Change permissions using GUI and CLI

## 💡 Hints & Tips
- Use `Get-Help` in PowerShell to learn about commands.
- Right-click Command Prompt to run as Administrator if needed.
- `Tab` autocompletes directory names in CMD and PowerShell.
